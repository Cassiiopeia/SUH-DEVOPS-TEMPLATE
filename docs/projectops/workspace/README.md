# Workspace — 코드 작업 격리 공간

`harness/WORKFLOW.md`의 Phase 4(개발)에서 생성되는 **코드 산출물**을 호스트 환경(레포 루트)과 격리해 두는 디렉토리다.

> **경로 단일 규칙**: 산출물은 항상 `docs/suh-template/` 우산 아래에 둔다 (`harness/WORKFLOW.md` §"산출물 경로 단일 규칙"). 레포 루트나 임의 위치에 코드 산출물·임시 폴더를 만드는 것을 금지한다 (전역 규칙 5: Zero-Contamination).

## 용도

- 빌드·프로토타입·실험 코드 등 **레포 본 소스에 직접 들어가지 않는** 코드 작업의 격리 공간.
- 실제 프로젝트 소스를 수정하는 작업은 이 폴더가 아니라 해당 소스 위치에서 직접 수행한다 (implement 철학: 코드 자체가 산출물).
