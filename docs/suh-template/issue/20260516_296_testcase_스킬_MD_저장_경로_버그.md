# ❗[버그][Skills] testcase 스킬 MD 파일이 프로젝트 루트에 저장됨

**라벨**: 작업전

---

## 📝 현재 문제점

`/testcase` 스킬 실행 시 생성되는 MD 파일이 다른 스킬들과 달리 프로젝트 루트(`testcase-[이슈번호]-[설명].md`)에 저장됨.

다른 스킬들(`analyze`, `plan`, `report` 등)은 `docs/suh-template/{skill_id}/` 경로에 저장하는데, testcase만 누락됨.

또한 `analyze`, `plan`, `design-analyze`, `ppt`, `refactor-analyze`, `review`, `troubleshoot` 스킬들이 경로 계산을 위해 `suh_template.cli get-output-path` CLI를 bash로 호출하고 있어, CLI 실패 시 경로를 잡지 못하는 리스크 존재.

## 🛠️ 해결 방안

- `testcase` 스킬 저장 경로를 `docs/suh-template/testcase/YYYYMMDD_{이슈번호}_{정규화된제목}.md`로 수정
- 나머지 7개 스킬도 CLI 호출 없이 경로를 직접 명시하는 방식으로 통일
- 이슈번호 없을 때 순번(`001`, `002`…) 자동 사용

## 📋 영향 범위
