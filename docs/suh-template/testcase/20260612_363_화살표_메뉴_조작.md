### TC-05: ESC 분기 — 확인 화면은 머무름(stay)

| 항목 | 내용 |
|------|------|
| 전제조건 | "이 정보가 맞습니까?" 확인 화면 (안내에 `ESC 머무르기` 표시) |
| 절차 | `ESC` 입력 |
| 기대 | 종료되지 않고 확인 화면에 그대로 머무름. 실제 취소는 `아니오, 취소` 항목을 직접 골라야 동작 |
| 결과 | |

### TC-07: ESC 분기 — 타입 선택은 기존값 유지 / 모드 선택은 종료

| 항목 | 내용 |
|------|------|
| 절차 | (1) 타입 선택 화면에서 `ESC` → 기존 타입 유지 확인 / (2) 최상위 모드 선택에서 `ESC` → 종료 확인 |
| 기대 | 타입 선택 ESC는 기존 값 유지(취소), 모드 선택 ESC는 `취소` 항목과 동일하게 마법사 종료 |
